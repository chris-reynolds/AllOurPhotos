Server Debugging

Within webstorm, use aopServer-build. Run debug is shift/F9.
This is the equivalent to: 
        cd c:\projects\AllOurPhotos\server\build        ! the javascript directory
        node aopServerMain.js
Note that unlike the client area, the build directory is kept up to date by the Typescript compiler.
